📘 AI-Powered Chatbot with Vue.js & Nest.js (AWS Bedrock)

🚀 Technologies Utilisées

🖼️ Front-end : Vue.js

⚙️ Back-end : Nest.js

🖥️ Installation et Lancement

🟢 1. Front-end (Vue.js)

cd frontend
npm install
npm run dev

L'application sera disponible sur : http://localhost:5173

🟠 2. Back-end (Nest.js)

cd backend
npm install
npm run start

L'API sera disponible sur : http://localhost:3000
